  ● "访客告知书" 标题是后端 system_config 表中 visitor_notice 字段的 HTML 内容，通过 <rich-text> 渲染。
   
  修改方式
     
  在后台管理面板 → 系统配置中找到「访客告知书」配置项，给标题标签加上内联样式：

  <!-- 之前 -->
  <h3>访客告知书</h3>

  <!-- 改成 -->
  <h3 style="text-align:center;font-size:36px;color:#1a73e8;">访客告知书</h3>

  <rich-text> 支持常见内联 CSS：color、font-size（px）、text-align、background-color 等。
